### 2. 🐍 Introduction to Python & Machine Learning
* ✅ Python Variables, Data Types & Operators
* ✅ Loops (For, While) & Conditionals

* ✅ Functional Programming & Lambda Functions

* ✅ **NumPy:** Arrays, Broadcasting, Linear Algebra Basics

* ✅ **Matplotlib:** Line, Scatter, Histogram, Bar & Pie plots

* ✅ Descriptive Statistics & Data Distributions
* ✅ Exploratory Data Analysis (EDA) & Data Cleaning

* ✅ Decision Trees (Entropy, Information Gain)
* ✅ Random Forest (Ensemble Learning)
* ✅ Support Vector Machines (SVM) & Kernels
* ✅ Clustering: K-Means, Hierarchical, DBSCAN
* ✅ Overfitting Control & Model Performance Comparison
* ✅ **Projects:** Student Performance Indicator, Credit Card Fraud Detection, Phishing Classifier

### 7. 🕸️ Deep Learning
* ✅ Perceptron Concept & Multi-Layer Perceptron (MLP)
* ✅ Backpropagation Intuition & Mathematics
* ✅ PyTorch: Tensors, Autograd, Computational Graph
* ✅ Optimization: Momentum, AdaGrad, RMSProp, Adam
* ✅ Regularization (L1/L2, Dropout) & Batch Normalization
* ✅ **Projects:** MNIST Digit Recognition, Customer Churn Prediction

### 8. 🖼️ CNN & RNN
* ✅ **CNN:** Foundations, Architectures & Math (PyTorch)
* ✅ Data Augmentation & Pre-trained Models
* ✅ **RNN:** Forward Propagation, LSTM, GRU
* ✅ Transformers & Self-Attention Mechanism
* ✅ **Projects:** Hate Speech Detection, Sentiment Analysis, Text Generation

### 9. 🗣️ Natural Language Processing (NLP)
